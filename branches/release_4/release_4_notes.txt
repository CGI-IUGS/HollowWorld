Here is a summary of changes made to release_4 of HollowWorld:

- Added a top level OGC package. 
- Added SWE Common Data Model 2.0 (http://www.opengeospatial.org/standards/swecommon)
- Added SWE Service Model 2.0 (http://www.opengeospatial.org/standards/swes)
- Added SWE Sensor Observation Service 2.0 Model (http://www.opengeospatial.org/standards/sos)
- Included Sensor Web Enablement 1.0 package, containing SensorML1.0 and O&M 1.0. 
- SensorML2.0 was not included, as it is still in pre-RFC.
